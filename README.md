# Port Scanner

## Description
This project scans a target IP address or domain and checks for open ports using Python.

## Features
- Scans ports from 1 to 1024
- Detects open ports
- Works with IP address or domain name

## How to run
python port_scanner.py
## Example Output

Enter target: scanme.nmap.org  
Port 22 is open  
Port 80 is open
